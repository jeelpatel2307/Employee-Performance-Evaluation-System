# Video Streaming Platform

This project implements a Video Streaming Platform in Java. It allows users to upload, view, and manage videos.

## Features

- **User Authentication**: Users can create accounts and log in to access the platform.
- **Video Upload**: Authenticated users can upload videos to the platform.
- **Video Streaming**: Users can stream and watch videos uploaded to the platform.
- **Search Functionality**: Users can search for videos based on title, category, or other metadata.
- **User Profiles**: Users have profiles where they can manage their uploaded videos and account settings.

## Getting Started

To run the Video Streaming Platform locally:

1. Clone this repository.
2. Compile the Java files.
3. Run the main application file.
4. Open a web browser and navigate to the local host URL provided.

## Technologies Used

- Java
- Spring Boot
- MySQL
- HTML/CSS
- JavaScript
